Accessibility is not just a feature, it's a necessity.

## Semantic HTML

Always use the correct HTML element for the job. Use `<button>` for buttons, not `<div>`.

## ARIA Labels

When visual cues aren't enough, use `aria-label` to describe elements to screen readers.
